Project: WebInterface

Project Description

This project defines a Web interface for the Ultimate web-site. It is based on the JSON (JavaScript Object Notation).
The main two classes here are:
  • UltimateResult.java - defines the standard of Ultimate output result, namely what kind of information will be sent back from Ultimate to the web-site in order to visualize the results on the web-site.
  • UltimateInterface.java - defines the web interface itself. Contains methods that perform GET(for examples)/POST(for Ultimate requests). This interface uses JSON to send data from Web-site to Ultimate and back. Can be accessed via http://server.tld/WebEclipseBridge/if
