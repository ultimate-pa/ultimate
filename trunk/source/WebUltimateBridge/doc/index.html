Project: WebsiteUltimateBridge

Project Description

This project describes Ultimate Toolchains, etc. for the projects WebsiteEclipseBridge and Website. After editing this project, it has to be exported as deployable plug-in and added to /Website/WebContent/WEB-INF/lib/ and /WebsiteEclipseBridge/WebContent/WEB-INF/eclipse/plugins

Where to add what

Tasks

Add a new value to the TaskNames enum and add the String representation for this new Task represented to the users in initTaskNames() .

Toolchains

Create a new Class which extends Toolchain to the package de.uni_freiburg.informatik.ultimate.website.toolchains . Afterwards add a reference in the actTCs Array in the class Tasks .
Before loading the new Toolchain, all constraints will be checked (i.e. String length, and uniqueness of identifiers) - if one is not fulfilled, the new toolchain will be skipped with a Logger message

Examples

Add the new Example file to the package resources.examples . Afterwards put the example description to the init() method in the class Examples . Before loading the new Example, all constraints will be checked (i.e. String length, and uniqueness of identifier in the description) - if one is not fulfilled, the new example will be skipped with a Logger message
