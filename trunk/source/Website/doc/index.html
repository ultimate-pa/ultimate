Project: Website

Project Description

This project is a simple servlet container which can be loaded e.g. on a Tomcat server. It holds the index.jsp file, which represents the websites interface to the user.
It depends on the project WebUltimateBridge, which is added to the lib folder as jar file.

ACE: Ajax Cloud Editor

Overview and Links

The editor used on the website. Supports syntax highlighting and is written in Java-Script.
  • Project website: ace.ajax.org
  • Can be downloaded here: Git Repository
  • Demo: Kitchen Sink

Build ace with Boogie syntax high-lighting

The following procedure was taken from here and slightly extended to add the Boogie highlighting.
Install npmjs on your system: curl http://npmjs.org/install.sh | sh

Then:
  1. run: git clone git://github.com/ajaxorg/ace.git
  2. run: cd ace
  3. run: git submodule update --init --recursive
  4. run: make clean
  5. run: npm install dryice@0.3.1
  6. copy boogie files to ./lib/ace/mode
  7. add '"boogie",' to Makefile.dryice.js where the other syntax high lightings are listed
  8. run: make build
Find the Boogie syntax highlighting files here

Note: The boogie syntax highlighting mode was sent to the developer(s) of ACE (i.e. fabian.jakobs@web.de <Fabian Jakobs>) so that they can add it to their repository and maintain it. This would make the above description obsolete and the provided and compressed default .js files!
